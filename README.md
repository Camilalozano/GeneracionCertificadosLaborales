Instalar librerías:
pip install pandas openpyxl python-docx

La hoja `SECOP_NoEstructurado` del archivo Excel debe incluir las columnas `url` y `entidad_adjudicataria`. El valor de `url` se agrega al final de cada certificado como enlace de validación de fuentes oficiales, y el valor de `entidad_adjudicataria` se usa en el campo `{nombre}` del texto del certificado.
